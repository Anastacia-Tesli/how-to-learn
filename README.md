
Одностраничный сайт "Научиться учиться"

Проект посвящен принципам обучения. Содержит описания методик обучения и ссылки на полезные источники. Для создания страницы использованы средства HTML и CSS, в том числе семантические теги, позиционирование элементов, flexbox-верстка, анимации.

В планах доработка проекта - подключение новых страниц сайта, использование дополнительных технологий JavaScript.
